Alibaba RFQ Scraper Assignment

This ZIP contains:
1. alibaba_rfq_scraper.py - Python script using Selenium to scrape RFQ listings from Alibaba.
2. alibaba_rfq_data.csv - Scraped RFQ data (sample format).
3. README.txt - Overview of the project.

Instructions:
- Install dependencies: selenium, chromedriver
- Run the script: python alibaba_rfq_scraper.py
- Output will be saved in alibaba_rfq_data.csv
